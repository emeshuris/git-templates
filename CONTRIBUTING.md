# Branching strategy:

* Type: github-flow
* Production Branch: master
* Workflow:
  * Clone repo or Pull master
  * Branch from master
  * Push to server
  * Create merge request to master
* Misc:
  * Approver approves
  * Submitter merges
  * Squash and delete branch on merge
